# CS 330 – Computational Graphics and Visualization Final Project

## Project Overview

This project is my final submission for CS 330: Computational Graphics and Visualization. The goal was to recreate a realistic 3D scene using modern OpenGL techniques while demonstrating concepts learned throughout the course. My scene represents a desk workspace built from primitive shapes and enhanced with textures, UV mapping, lighting, camera controls, and object transformations.

## How do I approach designing software?

I approach software design by first identifying the overall goal of the project and then breaking it into smaller, manageable components. ## What new design skills has your work on the project helped you to craft?

This project strengthened several design skills, including:

- Designing a complete 3D scene from simple geometric primitives

I also gained a better understanding of how small adjustments to scaling, rotation, and positioning greatly affect the overall appearance of a scene.

---

## What design process did you follow for your project work?

My design process followed these steps:

1. Planned the scene layout.
2. Identified the objects needed.
3. Selected primitive meshes for each object.
4. Positioned and scaled each object within the scene.
5. Applied textures using UV coordinates.
6. Added lighting and materials.
7. Tested and refined the scene until it appeared balanced and realistic.

Rather than trying to perfect everything at once, I built the project one object at a time and continuously refined the overall composition.
